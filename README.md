# Image-to-Spreadsheet Web Application Using OCR

**Convert real-world table images into structured, editable spreadsheet data.**

## Overview
This project implements an end-to-end computer vision pipeline to automatically extract table data 
from images (printed/handwritten) and convert it to editable spreadsheets.

## Key Features
Image preprocessing and deskewing  
Table detection (classical + deep learning models)  
Cell segmentation and OCR  
Interactive web interface with editable grid  
Export to CSV, Excel, JSON  

## Tech Stack
- **Backend:** Python, Flask, OpenCV, Tesseract OCR
- **Deep Learning:** TableNet, DETR Transformers
- **Frontend:** HTML, CSS, JavaScript
- **Database:** None (stateless processing)
